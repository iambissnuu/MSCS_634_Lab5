MSCS 634 - Lab 5  
Clustering using Hierarchical and DBSCAN Algorithms  

Overview  
This lab explores clustering techniques using Hierarchical Clustering and DBSCAN on the Wine dataset. The goal is to understand clustering behavior, parameter impact, and evaluation methods.

Methods Used  
- Hierarchical Clustering  
- DBSCAN  
- StandardScaler  
- PCA  

Evaluation Metrics  
- Silhouette Score  
- Homogeneity Score  
- Completeness Score  

Key Insights  
- Hierarchical clustering performed best with 3 clusters.  
- DBSCAN detected noise but was sensitive to parameters.  
- Dataset structure favors hierarchical methods.  

Challenges  
- DBSCAN parameter tuning  
- Handling noise in evaluation
